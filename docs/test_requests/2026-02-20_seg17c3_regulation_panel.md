## テスト手順

### 1. 法令制限タブが消えていること
- [ ] 物件編集画面を開く
- [ ] タブバーに「⚖️ 法令制限」タブがないこと
- [ ] タブ一覧: 所在地・周辺情報 / 基本情報 / 価格・取引 / 管理・費用 / 土地情報 / 画像情報 / 登記情報

### 2. 土地情報タブの表示確認
- [ ] 土地情報タブをクリック
- [ ] 黄色い「⚖️ 法令制限を自動取得（用途地域・建ぺい率等）」ボタンが表示される
- [ ] 法規制フィールド（用途地域、建ぺい率、容積率、防火地域等）がFieldGroupで表示される
- [ ] 手動でのフィールド編集が可能

### 3. RegulationPanelモーダルの表示
- [ ] 自動取得ボタンクリック → モーダルが開く
- [ ] モーダルにLeaflet地図が表示される（座標がある場合）
- [ ] 「🔍 この座標で法令制限を自動取得」ボタンが表示される
- [ ] 重要事項説明書チェックリストが表示される

### 4. 自動取得機能
- [ ] 座標がある物件で自動取得ボタンクリック → 取得中スピナー表示
- [ ] 取得成功で「○○を設定しました」メッセージ
- [ ] 取得した値は土地情報タブのフィールドに反映される
- [ ] 座標がない場合はエラーメッセージ表示

### 5. モーダル閉じる動作
- [ ] ×ボタンでモーダルが閉じる
- [ ] 「閉じる」ボタンでモーダルが閉じる
- [ ] モーダル外（オーバーレイ）クリックでモーダルが閉じる

### 6. 他タブへの影響なし
- [ ] 所在地・周辺情報タブ: 正常に表示される
- [ ] 基本情報タブ: 正常に表示される
- [ ] 価格・取引タブ: 正常に表示される
- [ ] 管理・費用タブ: 正常に表示される
- [ ] 画像情報タブ: 正常に表示される
- [ ] 登記情報タブ: 正常に表示される

### 7. 保存動作
- [ ] 土地情報タブのフィールドを変更 → 保存 → 正常に保存される
- [ ] チェックリストのチェック状態が保存される

- 法令制限タブが完全に消えていること
