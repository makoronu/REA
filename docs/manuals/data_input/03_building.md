# 建物情報

建物・居住・駐車場・管理の入力方法を説明します。

---

## 建物基本

### 建物構造

**これは何？**
建物の主要構造部分の材質です。

**入力ルール**
- 形式: ラジオボタン（1つ選択）
- 必須/任意: 任意

**選択肢**
| 選択肢 | 説明 |
|--------|------|
| 木造 | 木材で建てられた建物 |
| 鉄骨造 | 鉄骨（H鋼等）で建てられた建物 |
| RC造 | 鉄筋コンクリート造 |
| SRC造 | 鉄骨鉄筋コンクリート造 |
| 軽量鉄骨 | 軽量鉄骨造（アパートに多い） |
| ALC | 軽量気泡コンクリート造 |
| その他 | 上記以外 |

### 築年月

**これは何？**
建物が完成した年月です。

**入力ルール**
- 形式: 日付選択
- 必須/任意: 必須（公開時）

**入力手順**
① カレンダーアイコンをクリックしてください
② 年と月を選択してください

**正しい選択例**
✅ 「2020年3月」

⚠️ 注意: 築年数は自動計算されます

## 居住情報

**これは何？**

**入力ルール**
- 形式: 半角数字
- 必須/任意: 必須（マンション公開時）

---

### 向き

**これは何？**
バルコニーや主要窓の向きです。

**入力ルール**
- 形式: 選択（ドロップダウン）
- 必須/任意: 任意

**選択肢**
北 / 北東 / 東 / 南東 / 南 / 南西 / 西 / 北西

**入力手順**
① 「向き」の▼をクリックしてください
② 主な採光方向を選択してください

---

### 間取り部屋数

**これは何？**
部屋の数です（居室の数）。

**入力ルール**
- 形式: 半角数字
- 必須/任意: 必須（公開時）

**正しい入力例**
✅ 「3」（3LDKの「3」の部分）

---

### 間取りタイプ

**これは何？**
間取りの種類です。

**入力ルール**
- 形式: ラジオボタン（1つ選択）
- 必須/任意: 必須（公開時）

**選択肢**
| 選択肢 | 説明 |
|--------|------|
| R | ワンルーム（居室のみ） |
| K | キッチン付き |
| DK | ダイニングキッチン付き |
| LDK | リビングダイニングキッチン付き |
| SLDK | サービスルーム+LDK |
| その他 | 上記以外 |

**正しい選択例**
✅ 3部屋+リビングダイニングキッチン → 部屋数「3」、タイプ「LDK」

---

### 間取り詳細

**これは何？**
各部屋の広さと種類の詳細情報です。

**入力ルール**
- 形式: 専用入力フォーム
- 必須/任意: 任意

**入力手順**
① 「追加」ボタンをクリックしてください
② 部屋の種類（和室/洋室/LDK等）を選択してください
③ 広さ（帖/㎡）を入力してください
④ 必要に応じて複数の部屋を追加してください

**正しい入力例**
✅ LDK 16帖、洋室 6帖、洋室 5帖、和室 6帖

[SS: 間取り詳細入力フォーム]

---

## 管理情報

### 管理形態

**これは何？**
マンションの管理方式です。

**入力ルール**
- 形式: 選択（ドロップダウン）
- 必須/任意: 任意（マンションは入力推奨）

**選択肢**
| 選択肢 | 説明 |
|--------|------|
| 自主管理 | 住民が自分たちで管理 |
| 一部委託 | 一部の業務のみ管理会社に委託 |
| 全部委託 | すべての管理を管理会社に委託 |
