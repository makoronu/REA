## 備考・問題点

- 全テストSelenium自動テストで検証完了
- 用途地域: react-select使用、「指定なし」は12番目（最後）に表示
- 向き: HTML select使用、「該当なし」は10番目（最後）に表示
- 管理形態: HTML select使用、「該当なし（戸建等）」は5番目（最後）に表示
- 向き・管理形態はvisible_for設定により、マンション・アパート等の物件種別でのみ表示される

**注記**: 用途地域はAPIで13件返されるが、UIでは12件表示。「第二種低層住居専用地域」が表示されていない可能性あり（要確認）
