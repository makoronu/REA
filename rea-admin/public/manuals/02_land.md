# 土地情報

土地詳細・法規制・接道の入力方法を説明します。

---

## 法規制（自動取得）

### 用途地域

**これは何？**
その土地に建てられる建物の用途を制限する地域区分です。

**入力ルール**
- 形式: 複数選択
- 必須/任意: 任意

**選択肢**
| 選択肢 | 説明 |
|--------|------|
| 第一種低層住居専用地域 | 低層住宅のための地域。店舗や事務所は建てられません |
| 第二種低層住居専用地域 | 低層住宅が中心。小規模な店舗は可能 |
| 第一種中高層住居専用地域 | 中高層住宅のための地域 |
| 第二種中高層住居専用地域 | 中高層住宅が中心。必要な店舗等は可能 |
| 第一種住居地域 | 住居の環境を守る地域。大規模な店舗は不可 |
| 第二種住居地域 | 住居が中心。店舗・事務所も可能 |
| 準住居地域 | 幹線道路沿いで車関連施設も可能 |
| 近隣商業地域 | 近隣住民のための店舗・事務所が建てられる |
| 商業地域 | 銀行・映画館など商業施設が集まる地域 |
| 準工業地域 | 環境悪化の恐れがない工場が建てられる |
| 工業地域 | 工場が建てられる地域。住宅も可能 |
| 工業専用地域 | 工場専用。住宅は建てられません |
| 無指定 | 用途地域の指定がない |

**入力手順**
① 「法令調査」ボタンをクリックしてください
② 自動で取得されます
③ 複数の用途地域にまたがる場合は複数選択できます

[SS: 法令調査ボタン]

⚠️ 注意: 自動取得後は内容を確認してください。境界付近は誤差が出る場合があります。

### 建ぺい率

**これは何？**
敷地面積に対する建築面積の割合（%）です。建物を真上から見た面積の上限を決めます。

**入力ルール**
- 形式: 半角数字（%）
- 必須/任意: 任意

**正しい入力例**
✅ 「60」（60%の場合）
✅ 「80」（80%の場合）

**間違った入力例**
❌ 「60%」（%は不要）

⚠️ 注意: 法令調査ボタンで自動取得できます

---

## 土地詳細

### 土地面積

**これは何？**
土地の面積（㎡）です。

**入力ルール**
- 形式: 半角数字（㎡）
- 必須/任意: 必須（公開時）

**正しい入力例**
✅ 「150.25」（150.25㎡）

**間違った入力例**
❌ 「150.25㎡」（単位不要）
❌ 「約150」（「約」不要、正確な数値を入力）

## 接道

### 接道情報

**これは何？**
敷地に接している道路の詳細情報です。

**入力ルール**
- 形式: 専用入力フォーム
- 必須/任意: 必須（公開時）

**入力手順**
① 「追加」ボタンをクリックしてください
② 方向（東・西・南・北など）を選択してください
③ 道路種別を選択してください
④ 道路幅を入力してください
⑤ 接道間口（敷地が道路に接している長さ）を入力してください

**道路種別の選択肢**
| 選択肢 | 説明 |
|--------|------|
| 公道 | 国道・県道・市道など公共の道路 |
| 私道 | 個人が所有する道路 |
| 位置指定道路 | 特定行政庁から指定を受けた私道 |
| 開発道路 | 開発行為で作られた道路 |
| 2項道路 | 幅4m未満だが建築基準法上の道路として認められたもの |

**正しい入力例**
✅ 方向「南」、種別「公道」、幅「6」、間口「10」

[SS: 接道情報入力フォーム]

⚠️ 注意: 建築基準法では、幅4m以上の道路に2m以上接していないと建物が建てられません
