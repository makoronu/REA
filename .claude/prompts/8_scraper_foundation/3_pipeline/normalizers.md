# 正規化パーサー（Normalizers）

## やること
1. price.py: 金額パース（「2,980万円」→29800000）
2. area.py: 面積パース（「150.25㎡」→150.25、坪→㎡変換）
3. address.py: 住所分割（都道府県/市区町村/残り）
4. date_parser.py: 日付パース（「2020年3月築」→2020-03、和暦対応）
5. mapping.py: 表記ゆれ解決（master_options api_aliasesから読み込み）

## ルール
- 変換失敗時はNoneを返す（例外を投げない）
- mapping.pyはDB駆動（ハードコード禁止）
