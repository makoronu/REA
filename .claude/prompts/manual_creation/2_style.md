# 文章スタイル確認

## やること

1. 対象読者のレベルを確認
2. 文章スタイルを決定
3. 用語集の必要性を判断

## 文章ルール

### 文の書き方
- 1文30文字以内を目安
- 「〜してください」で統一
- 受動態より能動態

### 専門用語
- 初出時に（※説明）を付ける
- 用語集にも追加

### 操作説明
```
① 「保存」ボタンをクリックしてください
② 確認画面が表示されます
③ 「OK」をクリックしてください
```

### 警告・注意
```
⚠️ 注意: 〜すると、データが消えます
```

## 完了条件

- [ ] 対象読者レベル確認済み
- [ ] 文章スタイル決定済み
