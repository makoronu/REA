# マニュアル作成プロンプト

**対象**: 初心者・ITに不慣れな人向けの操作マニュアル

---

## 実行順序

```
→ 1_prepare.md   # 対象項目洗い出し
→ 2_style.md     # 文章スタイル確認
→ 3_write.md     # 本文作成
→ 4_review.md    # レビュー・修正
```

---

## 原則

1. **1文1意味**（詰め込まない）
2. **主語を省略しない**
3. **専門用語には注釈**（※で説明）
4. **操作は番号付きステップ**
5. **正しい例・間違った例を両方書く**

---

## 禁止

- 「など」「等」で逃げる
- 「適宜」「必要に応じて」
- 「ご確認ください」で終わる
- 抽象的な説明
