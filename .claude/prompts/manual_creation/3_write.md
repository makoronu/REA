# 本文作成

## やること

1. 各項目を以下のテンプレートで作成
2. 正しい例・間違った例を必ず書く
3. スクリーンショット挿入位置を指示

## 項目テンプレート

```markdown
### 項目名

**これは何？**
（1〜2文で説明）

**入力ルール**
- 形式:
- 文字数:
- 必須/任意:

**入力手順**
①
②
③

**正しい入力例**
✅ 「〇〇〇」

**間違った入力例**
❌ 「×××」（理由）

[SS: 〇〇の入力画面]
```

## 注意

- 1項目ずつ丁寧に
- 省略しない
- 曖昧な表現禁止

## 完了条件

- [ ] 全項目の説明完了
- [ ] 例が全て記載済み
- [ ] SS挿入位置指示済み
